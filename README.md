# Amazon Sales Data Analysis — Excel Dashboard

## Overview
Cleaned and analyzed Amazon sales data using Excel, building pivot tables 
and an interactive dashboard to surface key business insights.

## What I Did
- Cleaned raw data: removed duplicates, fixed data types, handled missing values
- Built 4 pivot tables: by Category, Brand, Discount Percentage, and High Discount items
- Designed a dashboard combining key charts for quick insights
